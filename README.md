# ШІ-асистент для Moodle

Цей репозиторій містить вихідний код платформи [Moodle 4.4](https://moodle.org/) з інтегрованим плагіном **AI Assistant** (**ai_assistant**), який додає блок штучного інтелекту для підтримки навчального процесу.

Плагін **ai_assistant** також доступний як окремий [репозиторій](https://github.com/ArtemDziuba/ai_assistant) і може бути встановлений незалежно у будь-який екземпляр Moodle. Інструкція для його встановлення знаходиться нижче (блок **Плагін ai_assistant**).

Функціонал плагіну:
- створення курсів на основі файлу (силабус, програма, методичні матеріали тощо) або текстового опису;
- внесення змін до курсів за допомогою чату з ШІ-асистентом;

## Плагін ai_assistant

Плагін **ai_assistant** знаходиться в окремому [репозиторії](https://github.com/ArtemDziuba/ai_assistant) і може бути встановлений незалежно у будь-який екземпляр Moodle.

### Встановлення з окремого репозиторію

Плагін складається з двох частин. Скопіюйте їх у відповідні папки вашого Moodle:

| Частина | Куди розмістити |
|---|---|
| Блок (UI) | `moodle/blocks/ai_assistant/` |
| Серверна частина | `moodle/local/ai_assistant/` |

Після копіювання файлів запустіть оновлення Moodle:

```bash
# Через браузер
https://your-moodle-url/admin/index.php

# Або через CLI
bin/moodle-docker-compose exec webserver php admin/cli/upgrade.php
```

## Вимоги для встановлення плагіну в наявний Moodle

- Moodle 4.4+
- PHP 8.1+

## Використання

Перед використанням необхідно вказати API_KEY від Google в налаштуваннях плагіну:

https://your-moodle-url/admin/settings.php?section=local_ai_assistant

### Створення курсу

Для автоматичного створення курсу перейдіть на звичну сторінку створення курсу:

https://your-moodle-url/my/courses.php

При натисканні на кнопку "Створити курс" Вам запропонує створення вручну або створення за допомогою ШІ-асистента. При виборі створення за допомогою ШІ-асистента можна завантажити файл, за допомогою якого буде створено курс, або ввести текстову інструкцію.

### Редагування курсу

Для допомоги з редагуванням курсу перейдіть на сторінку перегляду одного з Ваших курсів. Праворуч вгорі з'явиться кнопка для відкриття чату з ШІ-асистентом, де доступне редагування та створення нових секцій.

## Розробка

Для локальної розробки плагіну редагуйте файли безпосередньо у папці `moodle/blocks/ai_assistant/`. Після змін у PHP-коді очищайте кеш:

```bash
bin/moodle-docker-compose exec webserver php admin/cli/purge_caches.php
```

Після змін у `db/access.php`, `db/install.xml` або `version.php` — збільшуйте версію плагіну та запускайте upgrade:

```bash
bin/moodle-docker-compose exec webserver php admin/cli/upgrade.php
```
